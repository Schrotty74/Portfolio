# Portfolio

Die GitHub-Pages-Portfolioseite von [Martin Schrott](https://github.com/Schrotty74).

Sie stellt ausgewählte Projekte vor und verlinkt direkt zu den jeweiligen Repositories und, wo verfügbar, zu den laufenden Web-Apps.

## Lokal öffnen

`index.html` direkt im Browser öffnen. Es gibt keinen Build-Schritt und keine Abhängigkeiten.

## Veröffentlichung

GitHub Pages in den Repository-Einstellungen für den Branch `main` und das Verzeichnis `/(root)` aktivieren. Die Seite ist anschließend unter:

`https://schrotty74.github.io/Portfolio/`

erreichbar.
