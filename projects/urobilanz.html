MACOS APP + WEB APP

UroBilanz

A local logging and analysis tool for urine and fluid records, available as a portable web app and a native SwiftUI app for macOS.

Highlights

  • Import, merge and add urine, water and note entries.
  • Dashboard plus day, week, month and year views.
  • Exports for backups, CSV, JSON and macOS PDF reports.

Privacy

UroBilanz works locally and is not a medical diagnosis app. Public screenshots use demo data only.
