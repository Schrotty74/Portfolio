WEATHER FOR CITY · v1.2.2

Available now

Weather for City provides a responsive weather card for Mac and iPhone. Search for a city, choose the correct location when needed, and view current conditions in Celsius or Fahrenheit.

Install shortcut ↗ Release notes ↗

Highlights

  • Animated, responsive weather card for Mac and iPhone.
  • City lookup with a location picker for ambiguous results.
  • Current weather from the public Open-Meteo APIs, with no API key.
  • German interface for DACH regions; English elsewhere.
  • Independent shortcut folders, ready for future releases.

Privacy and verification

Every published shortcut includes its own privacy report. The repository also provides checksums, signed shortcut files and ZIP downloads so each release can be inspected and verified.
